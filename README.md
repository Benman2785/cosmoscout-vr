# Purpose

My changes are aimed at midigating motion- or cyber sickness while using Cosmoscout with a Icaros Pro.

# Installation / Prerequisits

Some configurations and scripts expect files to be at specific places.
To ensure that everything works right away, please make sure that the following contents are in the specified paths:

<ul>
  <li>Cosmoscout repository at 'C:\Repos\cosmoscout-vr'</li>
  <li>Icaros Driver at 'C:\Repos\CDM-v2.12.36.20-WHQL-Certified'</li>
  <li>Map server repository at 'C:\Repos\docker-mapserver'</li>
</ul>

[Icaros Driver](https://ftdichip.com/drivers/vcp-drivers/)

# Getting Started

There are scripts for starting Cosmoscout on either the earth or mars.
Execute the script, press enter and you should get booted into Cosmoscout.
